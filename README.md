Project Objective
The objective of this project is to analyze retail sales data using PostgreSQL and extract meaningful business insights from product, outlet, and location-level information. This analysis simulates real-world retail business decision-making using SQL.

Dataset Columns Contains header:
      
      Item_Identifier
      Item_Fat_Content
      Item_Type
      Item_Weight
      Outlet_Identifier
      Outlet_Establishment_Year
      Outlet_Size
      Outlet_Location_Type
      Outlet_Type
      Sales

This project answers key business questions such as:
      1.Which product category generates the highest revenue?
      2.Do customers prefer Low Fat or Regular products?
      3.Which outlet type performs best in terms of sales?
      4.Which outlet size contributes the most revenue?
      5.Which city tier is most profitable for the business?
      6.Which outlets are underperforming and need attention?
      7.Do older outlets perform better than newly established ones?
      8.Which products should be promoted based on sales performance?
      9.Does product weight have any relationship with sales?
      10.Where should the company expand its outlets for maximum profit?
